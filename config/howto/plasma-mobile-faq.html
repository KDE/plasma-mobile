Plasma Mobile - Frequently Asked Questions

The Appswitcher button in the top-left corner doesn't work, why?
Most likely, you are not using Desktop Effects. Currently, Plasma Mobile requires Desktop Effects switched on to work properly. You can enable Desktop Effects in System Settings, in the Section "Workspace Appearance and Behavior".

I don't have a network connection, what can I do?
In the system tray panel at the top edge of the screen, you can find the Network Management Widget. Make sure your system is set up to use Network Manager, configurable via Yast, in the section Network Devices | Network Settings.
